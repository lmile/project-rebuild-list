RosimWDC-400X

Parking lots online

Wireless parking lots occupancy detection sensor

Connect parking lots in a wide field
Flexible signal output for controllingsystem

WDC-400x Wireless Data Collector acts as a
gateway between data server and all the
wireless parking sensors to acquire the
occupancy information of each parking spaces.
As a gateway of the wireless parking sensors
network, WDC-400x provides RS-232/485 and
Ethernet data communication interface to relay
parking information data to user system, as
well as 16 OCs and 4 relay outputs to meet
requirements of controlling applications. Based
on ROSIM LPLRTM technology, WDC-400x
achieves up to 1500m ultra-long range wireless
communication performance providing a wide
area sensor connection ability. The
employment of this key technology makes it
possible to install only one data collector for
one application field, which lowers system
costs and, moreover, making system simplified
to improve stability and efficiency.
